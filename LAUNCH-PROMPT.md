# LAUNCH-PROMPT.md — Как запустить/продолжить работу

> Этот файл — промпт для Claude Desktop при открытии проекта в новой сессии.
> Версия шаблона: 2.0.0

---

## ПЕРВЫЙ ЗАПУСК (новый клиент)

```
Прочитай CLAUDE.md и onboarding/ONBOARDING-PROMPT.md.
Запусти полную распаковку нового клиента.
```

## ПРОДОЛЖЕНИЕ РАБОТЫ (клиент уже настроен)

```
Прочитай CLAUDE.md. Проверь memory/active-context.md —
что делали в прошлый раз. Покажи статус и жди задачу.
```

## ЭКСПРЕСС-ЗАПУСК (быстрый старт без полной распаковки)

```
Прочитай CLAUDE.md и onboarding/express-onboarding.md.
Задай 3 ключевых вопроса и начни производство контента.
```

## КОМАНДЫ ДЛЯ ОПЕРАТОРА

| Действие | Что делать |
|---|---|
| Новый клиент | Скопировать `fabrika-template-v2/` → переименовать → подключить как Project |
| Распаковка | Дать клиенту промпт «запусти распаковку» |
| Контент | После распаковки: «threads [тема]», «telegram [тема]» |
| Статус | «статус» — покажет % заполненности brand/ |
| Голос | «адаптируй голос» — voice-adapter применит к любому тексту |
| Аудит памяти | «ревизия памяти» — еженедельно |
| Brand fill | «brand fill [данные]» — заполнить brand/ из готовых данных |

## ЧЕКЛИСТ ПЕРЕД ЗАПУСКОМ

- [ ] Папка скопирована и переименована под клиента
- [ ] Подключена как Project в Claude Desktop
- [ ] CLAUDE.md прочитан Claude (автоматически при подключении)
- [ ] Клиент готов к интервью (40-60 мин для полной распаковки)
- [ ] Нужные пакеты активированы в CLAUDE.md (Визуал / Видео / SEO)

## ПАКЕТЫ: БЫСТРОЕ ПОДКЛЮЧЕНИЕ

Открыть CLAUDE.md → секция "АКТИВНЫЕ ПАКЕТЫ" → поставить [x]:

```markdown
- [x] Core — всегда активен
- [ ] Визуал — nano-banana, carousel, heygen
- [ ] Видео — youtube, reels, veo-video, video-editor
- [ ] SEO — seo-blog-writer, swipefile
```

## MCP ИНСТРУМЕНТЫ (опционально)

| Инструмент | Когда подключать |
|---|---|
| Obsidian | Если у клиента есть vault для хранения контента |
| NotebookLM | Если нужен ресёрч рынка/аудитории |
| Canva | При активации пакета «Визуал» |
| Apple Notes | Для быстрых захватов клиента |

## 🔒 СТАРТОВЫЙ ПРОМПТ ДЛЯ НОВОГО ПРОЕКТА В COWORK

Когда открываешь **новый проект** Фабрики — скопируй и вставь этот промпт первым сообщением:

```
Прочитай CLAUDE.md.

ВАЖНО: Ты работаешь ТОЛЬКО в папке этого проекта — той, которую я выбрал в Cowork.
Все файлы создаёшь только внутри неё (brand/, memory/, output/, skills/).
Не сохраняй ничего в другие папки на компьютере — ни ~/Documents, ни ~/CLAUDE FIL AI, нигде.

После прочтения CLAUDE.md — запусти онбординг: прочитай onboarding/ONBOARDING-PROMPT.md и начни распаковку клиента.
```

## ⚙️ НАСТРОЙКИ COWORK ПРИ ЗАПУСКЕ

При выборе папки в Cowork убедись:
1. **Выбрана папка конкретного клиента** — например `projects/client-slug/` или клонированная `fabrika-template-v2/`
2. **НЕ выбирай родительскую папку** типа `ОТДЕЛ КОНТЕНТА/` — Claude увидит лишние файлы и может запутаться
3. После выбора папки — вставляй стартовый промпт выше
