## ОБЗОР СИСТЕМЫ

### Что это
Шаблон-фабрика для быстрого развёртывания ИИ-системы производства контента под нового клиента. Базируется на плагине `fabrika-kontenta-ai` v2.3.0 (6 агентов, 34 скилла), адаптированном под Claude Desktop.

### Как работает
1. Оператор копирует папку `fabrika-template-v2/` клиенту
2. Подключает как Project в Claude Desktop
3. Даёт клиенту `ONBOARDING-PROMPT.md` — Claude запускает распаковку
4. За ~1 час заполняется brand/, настраивается голос, запускается производство

### Архитектурное решение: OpenClaw → Cowork mode

| Компонент | OpenClaw (было) | Cowork mode (стало) |
|---|---|---|
| Оркестрация | 6 Telegram-ботов + gateway | Agent() tool — параллельные субагенты |
| Конфигурация | openclaw.json | CLAUDE.md + settings.json |
| Интерфейс | Telegram | Cowork GUI (Desktop app) |
| Поиск по памяти | BM25 + vectors + rerank | Read .md файлов + Bash для анализа |
| Скиллы | OpenClaw skill API | Reference .md файлы |
| Выполнение | Последовательно | Параллельно через Agent() |
| Диагностика | openclaw doctor | Tech агент + Bash |
| Планировщик | Нет | Cowork scheduled tasks |
| Трекинг | Нет | TodoWrite виджет |

### Как подключать пакеты

При онбординге система спрашивает клиента:

```
Какие платформы используешь?
☐ Telegram         → ядро (уже включено)
☐ Threads          → ядро (уже включено)
☐ Instagram        → + пакет «Визуал»
☐ YouTube          → + пакет «Видео»
☐ Reels/Shorts     → + пакет «Видео»
☐ Блог/сайт        → + пакет «SEO»
```

Технически: папки пакетов уже в шаблоне, но CLAUDE.md содержит секцию `АКТИВНЫЕ ПАКЕТЫ`, которая обновляется при онбординге. Claude читает скиллы из пакета ТОЛЬКО если он активирован.

```markdown
## АКТИВНЫЕ ПАКЕТЫ
- [x] Core (всегда)
- [ ] Визуал
- [ ] Видео
- [ ] SEO
```

## РИСКИ И МИТИГАЦИЯ

| Риск | Вероятность | Митигация |
|---|---|---|
| Claude не читает CLAUDE.md автоматически | Низкая | Подключение как Project в Claude Desktop |
| Контекстное окно переполнено | Средняя | Ленивая загрузка: читать skills/ только по запросу |
| Клиент не проходит распаковку до конца | Средняя | Express-режим + возможность до-заполнить позже |
| Стилевые коррекции теряются между сессиями | Средняя | learning/corrections.md + CLAUDE.md читает при старте |
| personal-dna.md не заполнен → voice-adapter не работает | Средняя | voice-adapter читает voice-style.md как fallback |
| .plugin формат изменится | Низкая | Параллельный вариант — просто папка |
