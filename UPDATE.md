# Обновление проекта

## Для Натальи (одна команда)

Открой терминал в папке проекта и выполни:

```bash
git pull
```

Или попроси Claude: **«обнови проект»** — он выполнит `git pull` и покажет что изменилось.

## Что происходит

Олег загружает обновления (новый контент, правки, скиллы) в репозиторий.
Ты делаешь `git pull` — и получаешь всё свежее.

## Первый раз (клонирование)

Если у тебя ещё нет папки проекта:

```bash
git clone https://github.com/Jeezzzy/nataliya-andronova-marketing.git
```

Потом просто заходи в папку и делай `git pull` когда нужно.
