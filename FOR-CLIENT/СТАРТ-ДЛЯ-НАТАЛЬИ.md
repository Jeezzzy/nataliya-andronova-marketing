# Как работать со своим проектом через Claude Code

Привет, Наталья! Этот гайд объясняет как ты можешь управлять своим ботом и сайтом
без программиста — через общение с AI ассистентом Claude.

---

## Что такое Claude Code

Claude Code — это AI-ассистент, который работает прямо в папке твоего проекта.
Ты пишешь ему задачу обычными словами, он делает изменения в коде и деплоит на сервер.

**Примеры что можно попросить:**
- «Измени текст приветствия в боте»
- «Поправь цену тарифа Рост на 19 990»
- «Добавь новый вопрос в FAQ на сайте»
- «Задеплой изменения на сервер»

## Структура проекта (коротко)

Твой проект состоит из трёх частей:

**1. Лендинг** — сайт nataliya-andronova.ru
Все тексты сайта хранятся в одном файле `src/content.ts`.
Чтобы изменить текст на сайте — просто скажи Claude что именно поменять.

**2. Telegram-бот** — @nataliya_andronova_bot
Воронка прогрева 9 дней. Тексты — в файле `bot/src/text.ts`.
Голосовые сообщения — в папке `bot/assets/voice/`.

**3. Admin Panel** — панель управления
Заходи на https://nataliya-andronova.ru/admin/ чтобы смотреть статистику,
управлять пользователями и отправлять рассылки.

---

## Как начать работу

1. Установи Claude Code (технический специалист поможет — см. папку `setup/`)
2. Открой папку проекта в Claude Code
3. Пиши задачи обычным языком

Подробные примеры запросов → см. файл `КАК-УПРАВЛЯТЬ-БОТОМ.md`

---

## Важные ограничения

⚠️ **InSTART** — название внутренней программы. Не упоминать его публично: ни на сайте, ни в соцсетях, ни в рекламе. Только внутри бота.

⚠️ **Гарантия возврата** — в проекте её нет. Не добавлять это в тексты.

⚠️ **Тарифы** — финальные цены согласованы. Менять только после явного решения.
