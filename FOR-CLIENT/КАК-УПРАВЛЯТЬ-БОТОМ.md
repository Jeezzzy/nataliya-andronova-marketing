# Примеры запросов к Claude — бот и лендинг

Просто скопируй нужный запрос и отправь Claude (адаптируй под свою задачу).

---

## Изменить текст в боте

```
Измени приветственное сообщение в боте. Сейчас там написано [вставь текущий текст].
Замени на: [вставь новый текст]
После изменения задеплой на сервер.
```

```
В боте на день 3 прогрева поменяй текст для сегмента «декрет».
Новый текст: [вставь текст]
Задеплой.
```

---

## Заменить голосовые сообщения в боте

```
У меня есть новые голосовые файлы 1.ogg и 2.ogg в папке [укажи папку].
Замени ими первые два голосовых в боте (voice_01 и voice_02).
Задеплой бота на сервер.
```

---

## Изменить текст на сайте

```
На сайте в блоке Hero поменяй подзаголовок.
Сейчас: «20+ направлений — от SMM до нейросетей. 3 400+ выпускниц.»
Замени на: [новый текст]
Собери и задеплой сайт.
```

```
В FAQ добавь новый вопрос:
Вопрос: «[вопрос]»
Ответ: «[ответ]»
Задеплой.
```

---

## Изменить цены

```
Поменяй цену тарифа «Рост» на 19 990 ₽.
Обнови и в боте и на сайте — они должны совпадать.
Задеплой оба.
```

---

## Проверить статус бота

```
Проверь статус бота на сервере — все ли контейнеры работают.
Покажи последние логи бота.
```

---

## Перезапустить бота (если завис)

```
Бот не отвечает. Перезапусти контейнер na_bot на сервере.
```

---

## Добавить/удалить кейс на сайте

```
Добавь новый кейс ученицы в блок «Они тоже не верили»:
Имя: [имя]
Ситуация до: [текст]
Результат: [текст]
Задеплой сайт.
```

---

## Обновить гостевой ключ InSTART

```
Обнови гостевой ключ доступа в боте.
Новый ключ: [вставь ключ]
```

Ключ используется на День 7 воронки — когда пользователь нажимает «Хочу гостевой доступ».
Действует 48 часов после активации.

---

## Обновить лид-магнит (PDF)

```
У меня новый PDF «dekret_10k.pdf» в папке [путь].
Замени им старый лид-магнит для сегмента «декрет» в боте.
Задеплой.
```

---

## Полезные советы

- Claude знает весь проект — описывай задачу своими словами, он разберётся
- Всегда проси подтверждение перед деплоем если не уверена
- Если что-то пошло не так — напиши «откати последние изменения»
- Для сложных задач лучше делать по одному изменению за раз
