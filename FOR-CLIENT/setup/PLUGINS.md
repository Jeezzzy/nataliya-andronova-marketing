# Плагины Claude Code — проект Наталья Андронова

## Что такое плагин

Плагин расширяет возможности Claude Code: добавляет команды (`/commit`, `/copywriter`),
скиллы (инструкции для конкретных задач) и автоматизацию.

---

## Обязательные плагины

### `superpowers` (official)
**Зачем:** Базовый плагин разработки. Подключает планирование, TDD, дебаггинг, ревью кода.
Без него Claude работает хуже с кодом.
**Команды:** `/brainstorming`, `/systematic-debugging`, `/requesting-code-review`
**Установка:** Официальный, доступен в каталоге CC.

---

### `commit-commands` (official)
**Зачем:** Создаёт git-коммиты одной командой с правильным форматом сообщений.
**Команды:**
- `/commit` — сделать коммит
- `/commit-push-pr` — коммит + push + открыть PR
**Установка:** Официальный каталог CC.

---

### `feature-dev` (official)
**Зачем:** Помогает реализовывать новые фичи — анализирует архитектуру, строит план, пишет код.
Используй при добавлении нового функционала в бот или лендинг.
**Команды:** `/feature-dev`
**Установка:** Официальный каталог CC.

---

## Дополнительные (по задачам)

### `content-dept-fil-ai` (Fil AI Hub)
**Зачем:** Фабрика контента. Нужен если требуется писать посты, тексты, контент-план для Натальи.
**Команды:** `/coordinator`, `/copywriter`, `/marketer`, `/telegram`, `/threads`
**Установка:** Плагин из экосистемы Fil AI Hub (получить у Олега).

---

### `pr-review-toolkit` (official)
**Зачем:** Комплексное ревью кода перед деплоем — проверяет баги, безопасность, качество.
Используй перед значимыми изменениями в боте.
**Команды:** `/review-pr`
**Установка:** Официальный каталог CC.

---

### `claude-code-setup-kit` (Fil AI Hub)
**Зачем:** Настройка CC под конкретную роль. Запускается один раз при онбординге.
**Команды:** `/setup-claude`
**Установка:** Плагин из экосистемы Fil AI Hub.

---

## Как установить плагин

**Вариант 1 — через файл `.plugin`:**
Дважды кликни по файлу `.plugin` → CC откроет диалог установки.

**Вариант 2 — через каталог CC:**
В Claude Code открой каталог плагинов и найди нужный.

**Вариант 3 — команда `/setup-claude`** (если установлен `claude-code-setup-kit`):
Запустит интервью и поставит нужные плагины автоматически.

---

## Проверка установленных плагинов

В Claude Code выполни:
```
/status
```
Или спроси: «Какие плагины у тебя установлены?»
