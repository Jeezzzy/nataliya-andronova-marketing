# Чеклист настройки Claude Code — Наталья Андронова

Выполняй по порядку. Каждый шаг занимает 1-5 минут.

---

## Шаг 1 — Установка Claude Code

```bash
# Требуется Node.js 18+
npm install -g @anthropic-ai/claude-code

# Проверка
claude --version
```

Затем авторизуйся: запусти `claude` и следуй инструкции (вход через браузер).

---

## Шаг 2 — SSH-ключ для сервера

> Нужен для деплоя бота и лендинга на VPS.

Подробно: см. `SSH-SETUP.md`

Коротко:
```bash
# Скопировать ключ clients_vps в ~/.ssh/
# Установить права
chmod 600 ~/.ssh/clients_vps

# Проверить подключение
ssh -i ~/.ssh/clients_vps root@155.212.128.171 "echo OK"
```

---

## Шаг 3 — Установка плагинов

> Подробно какие и зачем — см. `PLUGINS.md`

Плагины устанавливаются двойным кликом по `.plugin` файлу или командой в CC.

**Минимальный набор для этого проекта:**

- [ ] `content-dept-fil-ai` — фабрика контента (если нужен контент)
- [ ] `commit-commands` — git коммиты одной командой (`/commit`)
- [ ] `superpowers` — обязательный базовый плагин разработки
- [ ] `feature-dev` — для работы с кодом проекта

**Опционально:**
- [ ] `pr-review-toolkit` — ревью кода перед деплоем

Установка через `/setup-claude` (если установлен `claude-code-setup-kit`).

---

## Шаг 4 — Открыть проект

```bash
# Перейти в папку проекта
cd "/путь/к/CLAUDE Devs/nataliya-andronova"

# Запустить Claude Code
claude .
```

Claude автоматически прочитает `CLAUDE.md` и загрузит контекст проекта.

---

## Шаг 5 — Проверка

После открытия проекта спроси Claude:
> «Прочитай CLAUDE.md и скажи что ты знаешь о проекте»

Если ответ содержит информацию о боте, лендинге, VPS и стеке — всё настроено правильно.

---

## Шаг 6 — Тест деплоя (опционально)

```bash
# Проверить доступ к серверу
ssh -i ~/.ssh/clients_vps root@155.212.128.171 "docker compose -f /opt/natalia/docker-compose.yml ps"
```

Должны быть видны 3 контейнера: `na_bot`, `na_backend`, `na_frontend`.
